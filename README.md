# S&C Bouw

Site voor S&C Bouw (aannemer, Maastricht — Nederland & België), 1‑op‑1 herbouw van de GoHighLevel preview op `https://preview.deprocesdesigners.nl/sc-home` in Next.js 14 + Tailwind.

## Dev

```bash
npm install
npm run download:assets   # eenmalig: download afbeeldingen naar public/images
npm run dev               # http://localhost:3022
```

## Build

```bash
npm run build && npm run start
```

## Stack

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS 3.4
- Fonts: Space Grotesk (display) + DM Sans (body), via `next/font/google`
